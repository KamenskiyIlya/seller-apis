# Обновление остатков и цен на маркетплейсах Яндекс и Ozon

В данном репозитории представлены 2 скрипта, которые позволят обновить остатки и цены у товаров, которые Вы продаете на маркетплейсах Яндекс и Ozon.

## Как работает скрипт seller

Данный скрипт помогает обновлять остатки и цены у Вашего асортимента в маркетплейсе Ozon.

Порядок работы скрипта:
1. В первую очередь скрипт берет информацию о Вашем личном кабинете продавца из файла `.env`. 
2. После этого получает все артикулы товаров из Вашего магазина на маркетплейсе. 
3. Получает информацию об остатках и ценах с сайта часовой компании [ООО "Группа АВГУСТ"](https://timeworld.ru/).
4. Формирует остатки в соответствии с остатками на складе компании после чего обновляет их на Ozon.
5. Формирует цены в соответствии с ценами компании после чего подгружает эти цены в Ozon

## Как работает скрипт market

Данный скрипт делает все тоже самое что и seller, но для Яндекс маркета и с небольшими различиями в порядке. Товар на Яндексе реализуется по двум моделям продаж: 
- DBS (Delivery by Seller)
- FBS (Fulfillment by Seller)

В связи с этим скрипт будет обновлять цены и остатки в Яндекс маркете для каждой модели продаж отдельно.

Порядок работы скрипта:
1. Скрипт берет информацию о Вашем личном кабинете продавца и информацию о складах из файла `.env`.
2. Получает информацию об остатках и ценах с сайта часовой компании [ООО "Группа АВГУСТ"](https://timeworld.ru/).
3. Получает все артикула товаров, которые продаются в Вашем магазине по модели продаж FBS.
4. Пока только для артикулов, которые продаются по FBS модели формирует остатки и цены в соответствии с остатками и ценами компании, после чего подгружает на Яндекс.
5. Дальше скрипт получает все артикула товаров, которые продаются по моедли DBS.
6. Для DBS продаж формирует остатки и цены в соответствии с остатками и ценами компании, после чего подгружает на Яндекс.
